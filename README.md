### 介绍

组件库模板

## 安装

```bash
npm i tomato -S
```

## 快速上手

```js
import Vue from 'vue';
import { Button } from 'tomato';
import 'tomato/lib/index.css';

Vue.use(Button);
```

tomato 也支持按需引入、CDN 引入等方式，详细说明见

## 浏览器支持

Tomato 2 支持现代浏览器以及 Android >= 4.0、iOS >= 8.0。
